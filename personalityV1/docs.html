The Loop

Four Phases, One Deliverable

Each discussion follows four phases. This section expands on each phase and which agents are active.

1. Clarify

Define the problem, capture requirements, and establish shared understanding. The Moderator sets the agenda; the Librarian gathers context; the Provocateur questions assumptions.

2. Explore

Generate diverse approaches without committing. The Creative proposes unconventional ideas; the Cynical identifies risks; the Librarian provides precedent.

3. Narrow

Converge on the best approach. The Pragmatist drafts concrete plans; the Cynical stress‑tests them; the Moderator resolves conflicts and calls decisions.

4. Execute

Deliver the artifact. The Pragmatist builds; the Moderator verifies completeness; the group ships.

Getting Started

Use the Framework

LLMPersonality is a prompt‑driven framework. Define your agents, set the topic, and let the orchestration loop run.

Defining Agents

Each agent has a role, personality traits, and phase activation rules. Here\'s a minimal example:

{
  "name": "Pragmatist",
  "role": "Turns ideas into working deliverables",
  "traits": ["Execution-focused", "Low-friction", "Concrete"],
  "active_phases": ["narrow", "execute"],
  "instruction": "Collapse ambiguity into action. Prefer simple, robust solutions."
}

Running a Discussion

The Moderator initiates the loop and manages phase transitions. Each phase produces specific outputs:

1. Clarify  → Requirements document, shared definitions
2. Explore  → List of proposed approaches, risk register
3. Narrow   → Selected approach, implementation plan
4. Execute  → Final deliverable, verification checklist

Key Principles

  • Role boundaries: Each agent speaks only from its role. The Cynical agent doesn\'t propose solutions; the Creative doesn\'t stress‑test them.
  • Phase gates: The Moderator declares phase transitions. No skipping ahead, no revisiting settled decisions without new evidence.
  • Deliverable focus: Every phase must produce a tangible output. Discussions that don\'t converge are failed discussions.
