About LLMPersonality

An open-source framework for orchestrating multi-agent discussions using LLMs with distinct personalities and roles.

Our Mission

LLMPersonality enables more effective human-AI collaboration by structuring discussions around distinct agent personas. Each agent brings a unique perspective, ensuring discussions are thorough, balanced, and actionable.

Rather than relying on a single LLM to navigate complex decisions, the framework orchestrates multiple agents that challenge, refine, and build upon each other's contributions.

How It Works

The framework operates through a structured loop:

  1. Clarify – Define the problem and establish shared understanding.
  2. Explore – Generate diverse perspectives and expand the solution space.
  3. Narrow – Weigh trade‑offs and converge on viable options.
  4. Execute – Translate decisions into concrete deliverables.

A human Moderator guides the process, while specialized agents contribute their unique strengths at each phase.

Contributing

LLMPersonality is open-source and welcomes contributions:

  • Submit agent personality profiles and example interactions
  • Improve the orchestration logic and phase transitions
  • Enhance the documentation and examples
  • Report issues and suggest new agent roles
